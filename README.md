Vibe Bot: AI-Powered Sentiment Exploration

Vibe Bot is an AI-driven web application designed to analyze customer reviews from e-commerce platforms like Amazon and extract meaningful sentiment insights. The system helps users quickly understand public opinion about products by classifying reviews into sentiments and highlighting top customer feedback.
